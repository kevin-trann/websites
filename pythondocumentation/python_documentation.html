Introduction

Python is a high-level, general-purpose programming language known for its simple, readable syntax and powerful capabilities. It was created by Guido van Rossum and released in 1991. Python emphasizes code readability and is often recommended as a first language for new programmers.

Thanks to its vast ecosystem of libraries and frameworks, Python is used in many fields — including web development, automation, data science, machine learning, and game development. Its cross-platform nature means the same Python code can run on Windows, macOS, and Linux without modification.

Basic Syntax

Python uses indentation instead of curly braces or keywords to define blocks of code. This makes the code easier to read and consistent. For example, all code inside a loop or function must be indented.

Here's how you print a message, declare variables, and write a simple if condition:

print("Hello, World!")  # Output text
x = 10
if x > 5:
    print("x is greater than 5")
Variables and Types

In Python, you don’t need to declare a variable's type — it’s determined at runtime. This makes Python flexible and fast to write, though you must be mindful of types during calculations or string operations. You can store integers, floats, strings, and booleans in variables directly.

You can also use type() to check what type a variable is. Python supports basic types like int, float, str, and bool. 

age = 10  #int
pi = 3.14  #float
name = "Luffy" #str
is_alive = True #bool

print(type(age))  # Output: class 'int'
Conditionals

Python uses if, elif, and else to make decisions in code. Each block must be properly indented. Conditions are built using comparison operators like ==, !=, >, <, >=, <=, and logical operators like and, or, and not.

  • == - returns True if equal
  • != - returns True if NOT equal
  • > - returns True if greater than
  • < - returns True if lesser than
  • >= - returns True if greater or equal than
  • <= - returns True if lesser or equal than

Conditionals allow your program to respond to different situations dynamically. They’re essential for creating interactive, decision-making code.

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
else:
    print("Keep studying!")
Loops

Loops are used to repeat code multiple times. Python supports two main types of loops: for and while. The for loop is ideal for iterating over sequences (like lists or strings), while while is best when you don’t know ahead of time how many times to loop.

You can use break to exit a loop early or continue to skip to the next iteration. Loops are fundamental for automation and repetitive tasks.

# for loop
for i in range(3):
    print("Round", i)

# while loop
count = 0
while count < 3:
    print("Count:", count)
    count += 1
